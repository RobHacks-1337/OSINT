# Some useful Websites:

- whatsmyname : Username lookup Tool
	> Used to search for social media sites matching with a username
	> https://whatsmyname.me/

- Shodan : Search Engine for Hackers
	> Used to check countries with most servers
	> https://www.shodan.io/

- Yandex : Reverse image searching
	> Used to find geolocation from images
	> https://yandex.com/

- Wigle : Wireless networking map
	> Used to check ssid using bssid and vice versa
	> https://wigle.net/ 

- Mapdevelopers : Google maps + Custom Tools 
	> Provides a google maps functionality with custom tools(like circle drawing, Area Calculator,...) to aid in OSINT.
	> https://www.mapdevelopers.com/

- Sourcing Games : Learn OSINT in fun way
	> Gamified way of learning and training OSINT skills
	> https://sourcing.games/
